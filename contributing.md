# Contribution Guidelines

Please read this before opening a PR.

## What belongs here

- Tools, platforms, or repos used for **education in the Philippines** (K-12, ALS, college, vocational, civil service prep, etc.).
- Something you would **personally recommend** to a teacher, student, or dev building for PH schools.

## What does not belong

- Generic global edtech with no PH angle.
- Paywalled products with no free tier unless they're genuinely unavoidable in PH (explain why).
- Your startup's landing page with no working product.
- AI-generated link dumps.

## Pull request checklist

- [ ] Link works.
- [ ] Description says **who it's for** and **what problem it solves** (one or two sentences, plain English).
- [ ] Entry is in the right section (or you added a section with a good reason).
- [ ] Alphabetical order within each section.
- [ ] No duplicate of something already listed.

## Description style

Bad:

> Revolutionary platform transforming the educational landscape through innovative AI-powered solutions.

Good:

> Auto-grader for CMSC 21-style intro programming. Used in 100+ PH schools; syncs grades to Moodle.
